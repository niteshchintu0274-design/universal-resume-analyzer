# Universal Resume Analyzer + Career Assistant

AI-powered resume analyzer with ATS scoring, job matching, resume rewriting, resume comparison, and career assistant chatbot.

## Features

- Resume PDF upload
- Resume text extraction
- OCR fallback support
- ATS score analysis
- Role and domain detection
- Resume heatmap
- Section-wise strengths and weaknesses
- AI resume rewriter
- Personalized 4-week roadmap
- Job matching demo UI
- Resume-based chatbot
- Resume version comparison

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Tailwind CSS
- SQLite
- Gemini AI support
- Adzuna Job API support
